# Custom Android Launcher

Кастомный Android лаунчер с возможностью скрытия приложений и блокировки сенсорного экрана.

## Функционал

### Основные возможности:
- ✅ Полная замена домашнего экрана и меню приложений Android
- ✅ Скрытие выбранных приложений
- ✅ Отключение сенсорного экрана по нажатию физических клавиш
- ✅ Поддержка наборов иконок
- ✅ Современный Material Design интерфейс
- ✅ Поиск приложений
- ✅ Выбор нескольких приложений одновременно

### Дизайн:
- **Цветовая схема:**
  - Фон: #212121 (темно-серый)
  - Акцент: #FFC700 (желтый/золотой)
  - Текст: #FFFFFF (белый)

## Требования

- Android SDK 26+ (Android 8.0 Oreo)
- Android Studio Arctic Fox или новее
- Kotlin 1.9.0+
- Gradle 8.0+

## Установка и сборка

### 1. Клонирование проекта
```bash
cd AndroidLauncher
```

### 2. Открыть в Android Studio
- Откройте Android Studio
- Выберите "Open an Existing Project"
- Выберите папку `AndroidLauncher`

### 3. Сборка APK
```bash
./gradlew assembleDebug
```

APK будет создан в: `app/build/outputs/apk/debug/app-debug.apk`

### 4. Установка на устройство
```bash
adb install app/build/outputs/apk/debug/app-debug.apk
```

## Использование

### Первый запуск:
1. После установки, нажмите кнопку "Home" на устройстве
2. Выберите "Custom Launcher" в качестве домашнего экрана
3. Нажмите "Всегда" для постоянного использования

### Настройка разрешений:
При первом запуске приложение запросит разрешение на отображение поверх других приложений. Это необходимо для функции блокировки экрана.

### Скрытие приложений:
1. Откройте меню приложений (нажмите на иконку приложений)
2. Выберите нужные приложения, нажав на них
3. Нажмите кнопку "СКРЫТЬ ВЫБРАННЫЕ"

### Настройка клавиш:
1. Перейдите в раздел "НАСТРОЙКА"
2. Нажмите "Изменить клавишу"
3. Нажмите нужную физическую клавишу (Volume Up/Down)

### Использование физических клавиш:
- Нажмите настроенную клавишу для скрытия/показа приложений
- При скрытии приложений автоматически блокируется сенсорный экран
- Повторное нажатие клавиши показывает приложения и разблокирует экран

## Основные компоненты

### MainActivity
- Главный экран лаунчера
- Отображение приложений в виде сетки
- Обработка нажатий физических клавиш
- Управление блокировкой экрана

### AppListActivity
- Список всех установленных приложений
- Выбор приложений для скрытия
- Поиск приложений
- Массовые операции

### SettingsActivity
- Настройка физических клавиш
- Выбор набора иконок
- Просмотр статуса системы

### TouchBlockService
- Фоновый сервис для блокировки экрана
- Создает невидимый оверлей поверх экрана
- Блокирует все сенсорные события

### AppRepository
- Получение списка установленных приложений
- Управление скрытыми приложениями
- Запуск приложений

## Разрешения

Приложение требует следующие разрешения:

- `QUERY_ALL_PACKAGES` - для получения списка всех установленных приложений
- `SYSTEM_ALERT_WINDOW` - для отображения оверлея блокировки экрана
- `EXPAND_STATUS_BAR` - для управления системной панелью

## Известные ограничения

1. **Блокировка физических клавиш:** Android не позволяет полностью перехватывать некоторые системные клавиши (например, Power). Используйте клавиши громкости.

2. **Разрешение на оверлей:** На некоторых устройствах (особенно с кастомными прошивками) может потребоваться дополнительная настройка разрешений.

3. **Иконки приложений:** Поддержка сторонних наборов иконок будет добавлена в будущих версиях.

## Troubleshooting

### Приложение не устанавливается как домашний экран:
- Убедитесь, что в манифесте указан правильный intent-filter с категорией HOME
- Перезагрузите устройство после установки

### Не работает блокировка экрана:
- Проверьте, что разрешение "Отображение поверх других приложений" предоставлено
- Некоторые производители ограничивают эту функцию - проверьте настройки безопасности

### Физические клавиши не работают:
- Используйте только клавиши громкости (Volume Up/Down)
- Убедитесь, что приложение находится на переднем плане при нажатии
